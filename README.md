# StudySpot

Ein modernes Frontend-Webprojekt zur Darstellung von UI-Design und grundlegenden Webentwicklungsfähigkeiten.

## Über das Projekt
StudySpot ist eine einfache Webanwendung, die Studierenden hilft, passende Lernorte wie Cafés, Bibliotheken und ruhige Plätze in der Nähe zu finden.

## Funktionen
- Modernes und sauberes Design
- Responsives Layout (für Desktop und Mobile)
- Übersichtliche Startseite
- Darstellung von Lernorten und Bewertungen
- Bereiche für Studierende und Cafébesitzer

## Verwendete Technologien
- HTML
- CSS
- Bootstrap
